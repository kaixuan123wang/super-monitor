关于 SDK Demo

本页面用于测试 pageView 自动埋点和页面间跳转的 breadcrumb 记录。

📖 项目说明

这是一个用于演示和测试前端监控 SDK 全部功能的示例项目。

  • 错误监控：自动捕获 JS 错误、Promise 拒绝、资源加载失败
  • 性能监控：采集 FP、FCP、LCP、CLS、TTFB 等 Web Vitals 指标
  • 网络监控：拦截 XMLHttpRequest 和 fetch 请求
  • 埋点功能：手动埋点、自动埋点（点击、浏览、离开、曝光）
  • 用户系统：匿名 ID / 登录 ID 管理、用户属性、超级属性
  • 面包屑：用户行为轨迹记录
